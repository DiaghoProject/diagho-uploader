## Installation

### Pré-requis

- Python 3.x

### Installation

```bash

git clone https://github.com/DiaghoProject/diagho-uploader.git

cd diagho-uploader

# Create venv
python -m venv env

# Activate venv
source env/bin/activate

# Install dependences
pip install -r requirements.txt

# Copy config file
cp config/.env.example config.env

```
